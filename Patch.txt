При создании пустого репозитория и клонировании его с GitHub в локальный репозиторий
создается мастер ветка с именем main. Ее нужно переименовать с помощью команды 
git branch --move main master
Затем сделать в локальном репозитории первый коммит и связать обьявить на удаленном репозитории
что мастер ветка теперь называется master.Это делается командой
 git push --set-upstream origin master
